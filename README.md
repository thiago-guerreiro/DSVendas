# DS Vendas

A aplicação consiste em exibir um dashboard a partir de dados fornecidos por uma API construída com Spring Boot.

## Link para versão web

https://vendas-ds.netlify.app/

# Tecnologias Utilizadas

## Back-end
- Java
- Sprint Boot

## Banco de Dados
- PostgreSQL

## Front-end
- React
- TypeScript
- Bootstrap

## Deplay
- Back-end: Heroku
- Front-end: Netlify
